需求：对输入的文本进行检测，输入后进行裁剪成标准的数值。即把“0000.1”、“0000010”、“100010.”等相对应地转换成“0.1”、“10”、“100010”。

正则已经测试过，根本不能进行匹配到。所以只能使用JavaScript来处理。

代码如下：

var sParamVal = '44super0林.0206.81';

/* 清除非数字非小数点字符*/
var rNumber = /[^\d^\.]+/g;
sParamVal = sParamVal.replace(rNumber, '');

/* 强制转化成浮点类型*/
var sFloatVal = parseFloat(sParamVal);
console.log(sFloatVal); // 440.0206
